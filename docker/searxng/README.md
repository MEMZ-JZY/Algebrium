# 本地 SearXNG

SigmaForge 仅访问绑定在 `127.0.0.1:8088` 的本地 SearXNG JSON API。后端会将可用的 HTTP/HTTPS 结果转换为带标题、域名和摘要的可引用来源。

SearXNG 的上游引擎可能偶发超时、触发验证码或被限流。若常规查询没有可用来源，后端会在同一台本地 SearXNG 上以 Bing 重试一次；两次均无结果时会向模型和界面返回明确的受控搜索故障，而不会静默返回空来源。

启动开发环境时，运行 `scripts/start-algebrium-dev.ps1` 会自动启动该服务。不要把该端口暴露到局域网或公网，也不要将此组件用作通用网页代理。
