# 数学评测基线目标

## 目标

建立可重复运行的 SigmaForge 数学离线评测基线，覆盖代数、微积分、线性代数和几何。评测验证每道固定题例能被映射到受限 CAS 或几何工具，并声明是否必须进行符号验证。

## 验收标准

- `bun run sigmaforge:eval` 运行 12 道题例，12/12 通过。
- 题例覆盖四个领域：代数 3 道、微积分 5 道、线性代数 3 道、几何 1 道。
- 每个题例保存输入、预期归一化结果、工具调用要求和 `requiresVerification` 声明。
- `bun test --timeout 30000 test/sigmaforge`、桌面端 `bun run build` 与 `git diff --check` 通过。

## 范围

本阶段只评测离线工具契约：请求被安全地转换为 SageMath 调用、模拟内核返回的固定结果被正确归一化，以及几何图元的结构正确。它不调用线上模型、不写入知识库、不包含物理模块。

## 已知缺口

当前题例不等同于真实 SageMath 端到端正确性评测。后续应在 Docker SageMath 就绪时，把同一批题例接入内核集成测试，并对模型回答增加步骤与引用评分。
